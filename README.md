## 📝 Sobre o Projeto
O **MangaImports** nasceu da necessidade de conectar entusiastas de cultura pop japonesa a obras físicas de difícil acesso no mercado nacional. O projeto consiste em uma plataforma dedicada de e-commerce focada em resolver os principais gargalos desse nicho: controle dinâmico de estoque flutuante, automatização de pagamentos digitais e canal direto de comunicação comunitária para sugestões de novos títulos.

Desenvolvido no contexto acadêmico de engenharia de software, o sistema serve como modelo analítico de melhores práticas de desenvolvimento, abrangendo desde o mapeamento de casos de uso complexos e diagramas comportamentais detalhados até as abordagens de persistência de dados. A aplicação entrega valor real ao usuário ao garantir um ambiente seguro de checkout e um ecossistema transparente onde a opinião do leitor dita as futuras importações do catálogo.

## ✨ Funcionalidades Principais
O sistema é estruturado em torno de 8 casos de uso primários:

- 📖 **Visualização de Catálogo Avançada (UC-01):** Consulta e filtragem preditiva de mangás importados disponíveis.
- 🛒 **Realização Prática de Compras (UC-02):** Criação e persistência do carrinho com checagem automática de estoque.
- 💳 **Processamento de Pagamento Integrado (UC-03):** Atualização automática de status baseado no retorno de Webhooks.
- 💬 **Avaliações Verificadas (UC-04):** Sistema de notas e comentários exclusivo para clientes que receberam a obra.
- 📥 **Solicitação de Inclusão de Catálogo (UC-05):** Canal para clientes sugerirem obras internacionais ausentes na loja.
- 📝 **Manutenção Cadastral Autônoma (UC-06):** Gerenciamento de metadados, preços e lotes de estoque pelo administrador.
- ⚙️ **Moderação e Gestão de Sugestões (UC-07):** Painel para o lojista aprovar ou reprovar os pedidos de novos mangás.
- 📦 **Rastreamento e Despacho de Pedidos (UC-08):** Inclusão de códigos de postagem e acompanhamento do ciclo de vida logístico.

## 🛠 Tecnologias Utilizadas

### 💻 Front-end

* **Framework/Biblioteca:** React v19
* **Linguagem/Superset:** TypeScript
* **Estilização:** Tailwind CSS
* **Gerenciamento de Estado:** Context API
* **Build Tool:** Vite

### 🖥️ Back-end

* **Linguagem/Runtime:** Java 17 (JDK)
* **Framework:** Spring Boot 3.x
* **Banco de Dados:** PostgreSQL 16
* **ORM / Query Builder:** Hibernate / JPA
* **Autenticação:** JWT (JSON Web Tokens) e Spring Security

### ⚙️ Infraestrutura & DevOps

* **Containerização:** Docker & Docker Compose
* **Cloud:** Vercel (Frontend) e Supabase / AWS (Database & Hosting)
* **CI/CD:** GitHub Actions
